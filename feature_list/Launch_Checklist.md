# 产品上线准备清单 (Launch Checklist)

## 1. 产品验收 (UAT - User Acceptance Testing)

- [ ] **PM 核心流程走查**:
  - [ ] 注册/登录 (手机号验证)
  - [ ] 首页浏览 (Banner 跳转, 推荐列表)
  - [ ] 视频播放 (播放/暂停, 进度拖拽, 全屏, 移动端手势)
  - [ ] 搜索功能 (综合/最新/最热排序)
  - [ ] 个人中心 (历史记录分组, 收藏夹)
- [ ] **UI/UX 还原度检查**: 对照设计稿检查字体、颜色、间距、移动端适配。
- [ ] **异常流程测试**: 断网提示、空数据状态页、加载失败重试。

## 2. 文档与物料准备 (Documentation & Materials)

- [ ] **产品介绍文档 (Product One-pager)**: 用于快速向他人介绍产品定位、核心功能和优势。
- [ ] **产品演示 PPT (Demo Deck)**: 用于向领导汇报或向客户演示。
- [ ] **用户操作手册 (User Manual)**: 图文并茂的功能使用指南。
- [ ] **常见问题解答 (FAQ)**: 针对用户可能遇到的问题提供标准回答。
- [ ] **版本更新说明 (Release Notes)**: 记录本次上线的核心功能点。

## 3. 运营内容配置 (Content Operations)

- [ ] **内容预埋**: 录入至少 20+ 个高质量视频，覆盖各个分类。
- [ ] **Banner 配置**: 准备 3 张移动端首页 Banner 图及跳转链接。
- [ ] **默认数据**: 配置热门搜索词、默认头像等。

## 4. 数据埋点与监控 (Data & Analytics)

- [ ] **核心指标确认**: 确认 DAU、播放量、完播率等指标的统计口径。
- [ ] **埋点验证**: 验证关键按钮（播放、收藏、搜索）的点击事件是否上报。

## 5. 合规与法务 (Compliance)

- [ ] **用户协议 & 隐私政策**: 确保内容最新（特别是手机号收集条款）。
- [ ] **免责声明**: 视频版权相关声明。

## 6. 技术与部署 (Technical & Deployment)

- [ ] **生产环境部署**: 确保代码已推送到生产服务器。
- [ ] **域名与证书**: HTTPS 证书是否有效。
- [ ] **浏览器兼容性测试**: 确认在 Chrome, Edge, Safari (iOS), Firefox 上的表现。
- [ ] **网络环境测试**: 如果是内网项目，确认目标用户网络能正常访问。

## 7. 分发与访问 (Distribution & Access) - _Web 版特有_

- [ ] **访问入口**: 制作网站链接的**二维码**，方便手机扫码访问。
- [ ] **"添加到主屏幕" 教程**: 既然不做 App，需要准备一份简单的教程，教用户如何把网页“添加到手机桌面”，获得类似 App 的体验。
- [ ] **短链接**: 如果网址很长，准备一个短链接。

## 8. 应急预案 (Contingency)

- [ ] **回滚策略**: 确认若上线失败，如何快速回退版本。
- [ ] **客服话术**: 针对已知遗留问题（Known Issues）准备好解释话术。

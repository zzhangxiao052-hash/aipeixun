# AI 培训平台功能清单

| 业务子系统          | 一级模块         | 二级模块     | 三级模块                  | 功能详细说明                                                                                       |
| :------------------ | :--------------- | :----------- | :------------------------ | :------------------------------------------------------------------------------------------------- |
| **AI 智能学习门户** | **首页/工作台**  | Banner 轮播  | 动态展示                  | 支持图片/视频轮播，展示最新活动、推荐课程或重要通知。                                              |
|                     |                  | 智能推荐流   | 个性化推荐                | 基于用户标签（如行政、财务、技术等）推荐相关视频课程。                                             |
|                     |                  |              | 筛选与分类                | 支持按部门/岗位标签筛选推荐内容。                                                                  |
|                     |                  | 前沿洞察     | 模块展示                  | 展示 AI 通识、行业趋势等前沿内容，支持标签筛选。                                                   |
|                     |                  | 效能跃升     | 模块展示                  | 展示办公提效、数据分析等实战技能内容，支持标签筛选。                                               |
|                     |                  | 场景创新     | 模块展示                  | 展示生活助手、创意写作等创新应用内容，支持标签筛选。                                               |
|                     |                  | 精彩评价     | 评价展示                  | 展示学员对课程的优质评价，包含用户头像、角色、评论内容及来源课程。                                 |
|                     | **视频学习中心** | 视频播放器   | 基础播放控制              | 支持播放/暂停、音量调节、全屏/退出全屏、进度拖拽。                                                 |
|                     |                  |              | 沉浸式体验                | 播放时背景虚化处理，提供沉浸式观看体验。                                                           |
|                     |                  |              | 快捷键支持                | 支持空格键播放/暂停，左右方向键快进/快退。                                                         |
|                     |                  |              | 进度记忆                  | 自动记录用户观看进度，每 5 秒自动保存，支持断点续播。                                              |
|                     |                  |              | 播放状态提示              | 视频加载、播放结束等状态的 Toast 提示。                                                            |
|                     |                  | 侧边栏导航   | 章节导航                  | 展示视频章节列表，支持点击跳转至指定时间点，当前播放章节高亮显示。                                 |
|                     |                  |              | 系列合集                  | 展示当前系列课程列表，显示观看进度与状态（播放中/已完成/未开始），支持点击切换视频。               |
|                     |                  | 互动工具栏   | 点赞/取消点赞             | 用户可对视频进行点赞操作，实时更新点赞数。                                                         |
|                     |                  |              | 收藏/取消收藏             | 用户可收藏视频，支持弹窗确认与直接取消收藏，实时更新收藏数。                                       |
|                     |                  |              | 分享                      | 支持视频分享功能（预留接口）。                                                                     |
|                     |                  |              | 更多操作                  | 预留更多操作入口。                                                                                 |
|                     |                  | 评论互动区   | 评论列表                  | 展示视频评论，支持按时间/热度排序（预留）。                                                        |
|                     |                  |              | 发表评论                  | 支持用户发表评论（预留接口）。                                                                     |
|                     |                  | 推荐视频     | 相关推荐                  | 侧边栏展示相关视频推荐，支持点击跳转。                                                             |
|                     |                  | 工具链集成   | 工具展示                  | 展示视频中提到的 AI 工具，提供直达链接。                                                           |
|                     | **个人中心**     | 观看历史     | 历史记录                  | 展示用户观看过的视频列表，包含进度条、上次观看时间、剩余时间。                                     |
|                     |                  |              | 清空历史                  | 支持一键清空观看历史，带二次确认。                                                                 |
|                     |                  |              | 断点续播                  | 点击历史记录可直接跳转至上次观看时间点。                                                           |
|                     |                  | 我的收藏     | 收藏列表                  | 展示用户收藏的视频课程。                                                                           |
|                     |                  | 个人设置     | 基本信息                  | 查看和编辑用户名、职位、个人简介，支持更换头像（UI）。                                             |
|                     |                  |              | 安全设置                  | 修改登录密码。                                                                                     |
|                     | **系统设置**     | 权限管理     | 组织架构树                | 可视化展示集团-公司-部门-团队的层级结构，支持展开/折叠。                                           |
|                     |                  |              | 节点管理                  | 支持添加子公司、部门、团队，支持重命名和删除节点。                                                 |
|                     |                  |              | 成员管理                  | 支持按部门筛选成员，支持搜索成员。                                                                 |
|                     |                  |              | 成员操作                  | 支持添加新成员、编辑成员信息（姓名、邮箱、角色、部门）、删除成员、禁用/启用成员。                  |
|                     |                  |              | 角色权限                  | 支持设置成员角色（普通用户、部门经理、超级管理员）。                                               |
|                     |                  | 可视化驾驶舱 | 全员覆盖率                | 展示全员覆盖率核心指标，包含环比趋势，直观呈现数字化培训的推广进度与覆盖范围。                     |
|                     |                  |              | 累计学习时长              | 展示平台累计学习时长核心指标，包含环比趋势，量化全员投入学习的时间成本与积极性。                   |
|                     |                  |              | 课程完课率                | 展示整体课程完课率核心指标，包含环比趋势，监控培训计划的执行深度与完成质量。                       |
|                     |                  |              | 资源调用次数              | 展示 AI 工具资源调用次数核心指标，包含环比趋势，衡量学员从“学”到“用”的转化效果。                   |
|                     |                  |              | 学习活跃度趋势            | 折线图展示每日课程播放量与活跃学习人数趋势，支持时间范围筛选。                                     |
|                     |                  |              | 部门效能矩阵              | 散点图展示各部门人均学习时长与资源调用率的关系，分析部门效能。                                     |
|                     |                  |              | 技能画像                  | 雷达图展示全员在智能写作、数据分析等技能维度的兴趣分布。                                           |
|                     |                  |              | 热门课程榜单              | 展示播放量最高的课程排名，包含分类、播放量、趋势标签。                                             |
|                     |                  |              | 全员覆盖率详情-核心指标   | 展示总人数、已激活账号数（激活率）、本月登录人数（月活率），直观掌握用户规模。                     |
|                     |                  |              | 全员覆盖率详情-趋势分析   | 折线图展示近 30 天日活趋势，监控平台活跃度变化。                                                   |
|                     |                  |              | 全员覆盖率详情-部门对比   | 条形图展示各部门激活率对比，分析各部门推广情况。                                                   |
|                     |                  |              | 全员覆盖率详情-人员明细   | 展示姓名、部门、账号状态、最后登录时间，支持多维度筛选。                                           |
|                     |                  |              | 全员覆盖率详情-激活提醒   | 支持对未激活或长期未登录人员一键发送激活/召回提醒。                                                |
|                     |                  |              | 累计学习时长详情-核心指标 | 展示全员人均学习时长，支持环比增长分析，衡量整体学习投入度。                                       |
|                     |                  |              | 累计学习时长详情-部门排行 | 条形图展示部门人均时长排行 Top 5，发现学习型组织标杆。                                             |
|                     |                  |              | 累计学习时长详情-时长榜单 | 展示学员学习时长排名、姓名、部门、总学习时长及最近一次学习时间，激励学员良性竞争。                 |
|                     |                  |              | 课程完课率详情-核心指标   | 分别展示基础认知完成率、效能跃升技能完成率，精准评估不同类型课程的学习效果。                       |
|                     |                  |              | 课程完课率详情-优质课程   | 展示最受欢迎课程 Top 5，包含完课率与好评率，识别优质内容。                                         |
|                     |                  |              | 课程完课率详情-痛点课程   | 展示“半途而废”课程 Top 5，包含完课率与痛点标签，辅助课程优化。                                     |
|                     |                  |              | 课程完课率详情-多维查看   | 支持切换视角，既可按人员查看个人完课情况，也可按课程查看整体完课进度。                             |
|                     |                  |              | 资源调用详情-核心指标     | 展示资源点击总数、提示词复制总数、人均调用次数，量化 AI 工具的实际应用热度。                       |
|                     |                  |              | 资源调用详情-热门工具     | 条形图展示热门 AI 工具排行榜，了解学员工具偏好。                                                   |
|                     |                  |              | 资源调用详情-偏好分析     | 饼图展示复制提示词的类别占比，分析学员业务场景需求。                                               |
|                     |                  |              | 资源调用详情-人才挖掘     | 展示学员资源调用详情，包含姓名、部门、调用总次数、偏好领域及“效能跃升派”等画像标签，辅助人才发掘。 |

# 如何将网页添加到手机主屏幕 (Add to Home Screen Guide)

既然我们的产品是 Web 网页版，为了方便您像使用 App 一样快捷访问，建议您将网页“添加到主屏幕”。

## 🍎 iOS (iPhone/iPad) 用户

1.  **打开 Safari 浏览器**: 使用 Safari 打开我们的产品网址。
2.  **点击“分享”按钮**: 在屏幕底部中间找到分享图标（一个方框带个向上箭头的图标 🛫）。
3.  **找到“添加到主屏幕”**: 在弹出的菜单中向上滑动，找到并点击 **“添加到主屏幕” (Add to Home Screen)** 选项（通常是一个加号图标 ➕）。
4.  **确认添加**: 在右上角点击 **“添加”**。
5.  **完成**: 现在您的桌面上会出现一个新的图标，点击即可全屏访问，体验和原生 App 一致。

---

## 🤖 Android (安卓) 用户

_(以 Chrome 浏览器为例，其他浏览器操作类似)_

1.  **打开 Chrome 浏览器**: 使用 Chrome 打开我们的产品网址。
2.  **点击菜单按钮**: 点击屏幕右上角的三个点图标 (⋮)。
3.  **点击“添加到主屏幕”**: 在菜单中找到并点击 **“添加到主屏幕” (Add to Home Screen)** 或 **“安装应用”**。
4.  **确认添加**: 在弹出的确认框中点击 **“添加”**。
5.  **完成**: 图标会自动出现在您的桌面上。

---

## 💡 为什么这样做？

- **一键直达**: 无需每次输入网址或在微信里翻找链接。
- **沉浸体验**: 隐藏浏览器地址栏和底部工具栏，获得全屏的专注学习体验。

# 手机端原型说明

## 概述
本文件夹包含AI培训平台的手机端原型页面。手机端作为PC端的二级页面存在，通过PC端导航栏的"手机端预览"按钮进行访问。

## 文件结构
```
mobile/
├── MobileHome.jsx        # 手机端首页（包含推荐课程、快捷入口等）
├── MobileProfile.jsx     # 个人中心页面
├── MobileCourses.jsx     # 我的课程页面
└── README.md            # 本说明文档
```

## 主要功能

### 1. MobileHome.jsx - 手机端首页
- **顶部导航栏**：Logo、搜索、通知
- **Banner区域**：展示平台标语
- **快捷入口**：我的课程、学习历史、我的收藏、学习统计
- **分类导航**：全部、前沿洞察、效能跃升、场景创新
- **推荐课程列表**：展示课程卡片，包含进度条
- **底部导航栏**：首页、课程、分类、我的
- **返回PC端按钮**：固定在右上角，可随时返回PC端

### 2. MobileProfile.jsx - 个人中心
- **用户信息卡片**：头像、姓名、职位
- **学习统计**：学习时长、完成课程、学习积分
- **功能菜单**：
  - 我的课程
  - 我的收藏
  - 观看历史
  - 学习统计
  - 设置
  - 退出登录

### 3. MobileCourses.jsx - 我的课程
- **搜索功能**：搜索课程
- **筛选标签**：全部、学习中、已完成
- **课程卡片**：
  - 课程缩略图
  - 课程标题和分类
  - 学习进度条
  - 课程时长
  - 学习状态（已完成/继续学习/开始学习）

## 路由配置
手机端页面的路由都以 `/mobile` 开头：
- `/mobile` 或 `/mobile/home` - 首页
- `/mobile/profile` - 个人中心
- `/mobile/courses` - 我的课程

## 访问方式
1. **从PC端访问**：点击PC端导航栏右上角的"手机端预览"按钮
2. **直接访问**：在浏览器地址栏输入 `http://localhost:5173/mobile`
3. **从手机端返回PC端**：点击页面右上角的"返回PC端"按钮

## 设计特点
1. **独立性**：手机端页面不显示PC端的导航栏、Footer和反馈侧边栏
2. **响应式设计**：采用移动优先的设计理念，适配手机屏幕
3. **统一风格**：保持与PC端一致的品牌色彩和设计语言
4. **便捷导航**：底部固定导航栏，方便快速切换页面

## 技术栈
- React + React Router
- Tailwind CSS
- Lucide React Icons

## 注意事项
1. 所有手机端页面都包含"返回PC端"按钮，确保用户可以随时切换
2. 手机端页面使用独立的布局，不继承PC端的全局导航
3. 路由以 `/mobile` 开头，便于在App.jsx中统一处理布局逻辑
